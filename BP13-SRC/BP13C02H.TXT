      *==========================================================*              
      * 1. OBJECTIVE  : THIS PROGRAM WILL CHECK THE DBSS REQUEST *              
      *                 CREATED THROUGH INTERNET WITH (BP13FB78) *              
      *                 WITH THE REQUEST FILE (BP13K730) THAT    *              
      *                 MAY HAVE UPDATED DATA THROUGH ONLINE     *              
      *                 PANEL.                                   *              
      *                 IF THERE ARE CHANGES, PROGRAM WILL       *              
      *                 OUTPUT FILE AND RE-SEND TO DBSS DEVELOPER*              
      *                 BP13K730 - ALLOSCH, ELIG-SCH             *              
      *                 BP13K825 - PERSONAL DATA                 *              
      *                 BP13K830 - ADDRESS                       *              
      *                                                          *              
      * 2. INPUT      : BP13FB78 - BP13.FB78.DBSREQ              *              
      *                 BP13K730 - BP13.K730.MATURERQ            *              
      *                 BP13K825 - BP13.K825.REQOCCP             *              
      *                 BP13K830 - BP13.K830.ADDR                *              
      *                                                          *              
      * 3. OUTPUT     : P13FB78A - CONTAINS ALL RECORDS WITH OR  *              
      *                            WITHOUT UPDATES               *              
      *                 P13FB78B - CONTAINS RECORDS THAT IS      *              
      *                            UPDATED ONLY                  *              
      *                 P13FB78C - CONTAINS ORIGINAL VALUES OF   *              
      *                            THE UPDATED RECORDS ONLY      *              
      *==========================================================*              
      * MODIFICATIONS :                                          *              
      *                                                          *              
      * CHGE-NO  DATE       OIC  DESCRIPTION                     *

      *************************************************************             
       1000-OPEN-ROUTINE.                                                       
      *************************************************************             
           OPEN INPUT  BP13FB78                                                 
                       BP13K730                                                 
                       BP13K825                                                 
                       BP13K830                                                 
                OUTPUT P13FB78A                                                 
                       P13FB78B                                                 
                       P13FB78C.                                                
                                                                                
           IF WS-K730-STATUS NOT = ZEROES AND 97                                
              DISPLAY 'BP13K730 OPEN ERROR ' WS-K730-STATUS                     
              MOVE WS-K730-STATUS        TO RETURN-CODE                         
                                                                                
              PERFORM 9000-CLOSE-ROUTINE THRU 9000-EXIT                         
           END-IF.                                                              
                                                                                
           IF WS-K825-STATUS NOT = ZEROES AND 97                                
              DISPLAY 'BP13K825 OPEN ERROR ' WS-K825-STATUS                     
              MOVE WS-K825-STATUS        TO RETURN-CODE                         
                                                                                
              PERFORM 9000-CLOSE-ROUTINE THRU 9000-EXIT                         
           END-IF.                                                              
                                                                                
           IF WS-K830-STATUS NOT = ZEROES AND 97                                
              DISPLAY 'BP13K830 OPEN ERROR ' WS-K830-STATUS                     
              MOVE WS-K830-STATUS        TO RETURN-CODE                         
                                                                                
              PERFORM 9000-CLOSE-ROUTINE THRU 9000-EXIT                         
           END-IF.                                                              
                                                                                
           MOVE FUNCTION CURRENT-DATE(1:8)                                      
                                         TO WS-CURRENT-DATE.                    
                                                                                
           STRING WS-CURR-DATE(7:2) '/'                                         
                  WS-CURR-DATE(5:2) '/'                                         
                  WS-CURR-DATE(1:4)                                             
                  DELIMITED BY SIZE                                             
             INTO WS-RUN-DATE.                                                  
                                                                                
       1000-EXIT.                                                               
            EXIT.                                                               
                                                                                
                                                                                
      *************************************************************             
       1100-READ-BP13FB78.                                                      
      *************************************************************             
           READ BP13FB78                                                        
             AT END                                                             
                MOVE 'Y'                 TO WS-FB78-EOF                         
            NOT AT END                                                          
                ADD  1                   TO WS-FB78-READ-CNT.                   
                                                                                
       1100-EXIT.                                                               
            EXIT.                                                               
                                                                                
                                                                                
      *************************************************************             
       2000-PROCESS-FILES.                                                      
      *************************************************************             
           PERFORM 2050-INITILIZE-VARS THRU 2050-EXIT.                          
                                                                                
           MOVE 'N'                      TO WS-APPL-CHK                         
                                            WS-OCCP-CHK.                        
                                                                                
           MOVE SPACES                   TO BP13K730-REC.                       
           INITIALIZE                       BP13K730-REC.                       
                                                                                
           MOVE FB78-KEY-FLD             TO K730-KEY-NEW.                       
                                                                                
           READ BP13K730.                                                       
                                                                                
           EVALUATE WS-K730-STATUS                                              
               WHEN 00                                                          
                    ADD  1               TO WS-K730-READ-CNT                    
                    MOVE BP13FB78-REC    TO WS-P13FB78C-REC                     
                                                                                
                    IF K730-CDE-ALLOC-SCH NOT = FB78-CDE-ALLOC-SCH              
                       MOVE K730-CDE-ALLOC-SCH                                  
                                         TO FB78-CDE-ALLOC-SCH                  
                       MOVE 'U'          TO FB78-NUM-STATUS                     
                    END-IF                                                      
                                                                                
                    IF K730-CDE-ELIG-SCH NOT = FB78-CDE-ELIG-SCH                
                       MOVE K730-CDE-ELIG-SCH                                   
                                         TO FB78-CDE-ELIG-SCH                   
                       MOVE 'U'          TO FB78-NUM-STATUS                     
                    END-IF                                                      
                                                                                
                    PERFORM 2060-MOVE-NRICS  THRU 2060-EXIT                     
                                                                                
                    PERFORM 2070-COMPARE-APPL-NRIC                              
                    VARYING WS-CTR FROM 1 BY 1 UNTIL WS-CTR > 4                 
                                                                                
                    IF FB78-NUM-STATUS NOT = 'U'                                
                       PERFORM 2100-COMPARE-NRICS  THRU 2100-EXIT               
                    END-IF                                                      
                    PERFORM 4000-READ-BP13K830  THRU 4000-EXIT                  
                    PERFORM 6000-CHECK-BP13K730 THRU 6000-EXIT                  
                                                                                
                    IF FB78-NUM-STATUS = 'C' OR 'U'                             
                       WRITE WS-P13FB78C-REC                                    
                                                                                
                       ADD  1            TO WS-FB78C-WRITE-CNT                  
                                                                                
                       WRITE WS-P13FB78B-REC FROM BP13FB78-REC                  
                                                                                
                       ADD  1            TO WS-FB78B-WRITE-CNT                  
                    ELSE                                                        
                       ADD  1            TO WS-REC-NO-CHG-CNT                   
                    END-IF                                                      
                                                                                
                    WRITE WS-P13FB78A-REC FROM BP13FB78-REC                     
                                                                                
                    ADD  1               TO WS-FB78A-WRITE-CNT                  
               WHEN 10                                                          
               WHEN 23                                                          
                    DISPLAY 'NOT FOUND IN K730: ' FB78-KEY-FLD                  
                    ADD  1               TO WS-K730-NOTFND-CNT                  
                                                                                
                    WRITE WS-P13FB78C-REC FROM BP13FB78-REC                     
                                                                                
                    ADD  1               TO WS-FB78C-WRITE-CNT                  
                    MOVE 'C'             TO FB78-NUM-STATUS                     
                                                                                
                    WRITE WS-P13FB78A-REC FROM BP13FB78-REC                     
                                                                                
                    ADD  1               TO WS-FB78A-WRITE-CNT                  
                                                                                
                    WRITE WS-P13FB78B-REC FROM BP13FB78-REC                     
                                                                                
                    ADD  1               TO WS-FB78B-WRITE-CNT                  
               WHEN OTHER                                                       
                    DISPLAY 'BP13K730 READ ERROR ' WS-K730-STATUS               
                    DISPLAY 'K730 KEY: ' K730-KEY-NEW                           
                                                                                
                    MOVE WS-K730-STATUS  TO RETURN-CODE                         
                                                                                
                    PERFORM 9000-CLOSE-ROUTINE THRU 9000-EXIT                   
           END-EVALUATE.                                                        
                                                                                
           PERFORM 1100-READ-BP13FB78 THRU 1100-EXIT.                           
                                                                                
       2000-EXIT.                                                               
            EXIT.

       2060-MOVE-NRICS.

           MOVE FB78-NUM-OCCUP2-UIN   TO WS-FB78-OCCUP-NRIC(2).

             IF WS-FB78-OCCUP-NRIC(WS-CTR) NOT = SPACES AND LOW-VALUES          
                ADD  1      TO WS-FB78-OCCP-CNT                                 
             END-IF

      *************************************************************             
       2070-COMPARE-APPL-NRIC.                                                  
      *************************************************************             
                                                                                
           IF WS-K730-NRIC(WS-CTR)(1:2) NOT = '##'                              
              IF (WS-FB78-NRIC(WS-CTR) = SPACES OR LOW-VALUES OR ZEROES)        
              OR (WS-K730-NRIC(WS-CTR) NOT = WS-FB78-NRIC(WS-CTR))              
                  MOVE 'U'          TO FB78-NUM-STATUS                          
              END-IF                                                            
           END-IF.                                                              
                                                                                
           IF WS-K730-NRIC(WS-CTR)(1:2) = '##' AND                              
              WS-FB78-NRIC(WS-CTR) NOT = SPACES AND LOW-VALUES AND              
                                         ZEROES                                 
              MOVE 'U'          TO FB78-NUM-STATUS                              
           END-IF.                                                              
                                                                                
       2070-EXIT.                                                               
            EXIT.                                                               
                                                                                
                                                                                
      *************************************************************             
       2100-COMPARE-NRICS.                                                      
      *************************************************************             
           MOVE 'Y'                      TO WS-APPL-CHK.                        
                                                                                
           PERFORM VARYING WS-IDX1 FROM 1 BY 1                                  
             UNTIL WS-IDX1 > 4                                                  
                OR WS-FB78-APPL-NRIC(WS-IDX1) = SPACES OR LOW-VALUES            
                   PERFORM 3000-READ-BP13K825 THRU 3000-EXIT                    
           END-PERFORM.                                                         
                                                                                
           MOVE 'N'                      TO WS-APPL-CHK.                        
           MOVE 'Y'                      TO WS-OCCP-CHK.                        
                                                                                
           PERFORM VARYING WS-IDX1 FROM 1 BY 1                                  
             UNTIL WS-IDX1 > 8                                                  
                OR WS-FB78-OCCP-NRIC(WS-IDX1) = SPACES OR LOW-VALUES            
                   PERFORM 3000-READ-BP13K825 THRU 3000-EXIT                    
           END-PERFORM.                                                         
                                                                                
           MOVE 'N'                      TO WS-OCCP-CHK.                        
                                                                                
       2100-EXIT.                                                               
            EXIT.

           EVALUATE FB78-NUM-APPLT1-RACE                                        
               WHEN 'C  '                                                       
                    MOVE '0'             TO WS-FB78-APPL-ETH(1)                 
               WHEN 'M  '                                                       
                    MOVE '5'             TO WS-FB78-APPL-ETH(1)                 
               WHEN 'I  '                                                       
                    MOVE '6'             TO WS-FB78-APPL-ETH(1)                 
               WHEN 'O  '                                                       
                    MOVE '7'             TO WS-FB78-APPL-ETH(1)                 
               WHEN OTHER
